# Звонить некому

**2026-05-22**

## Что обсуждали

Так как команда состоит из одного человека, то и звонить некому.
Тем не менее у меня есть некоторые мысли.

## Что решили

- Приложение будет написано на актуальной версии Angular с использованием последней версии Angular Material
- Для будущей авторизации будет использоваться Firebase
- Использовать Figma Design как опору для разработки UI
- Коммиты дневников не сквошить.
- Сквошить нужно все коммиты разработки.
- По возможности каждый PR будет содержать ссылку на issue
- Необходимо изучить работу Dependencies Injection, Strategy и changeDetection

## Поручения и зоны ответственности

- Репозиторий : Я уже создал репозиторий с базовой структурой директорий и конфигами (Angular, ESLint, Prettier, Stylelint, Lint-Staged, Husky)

## Запланировано

- Внесение изменений в структуру проекта
- Создание первых базовых компонентов (layout, header, footer)
- Изучение работы с Angular Material
- Провести атомизацию проекта на компоненты

# Всё идёт по по-плану

**2026-06-22**

## Что обсуждали

На этой неделе созвонов с ментором не было, но был созвон с интервьюером Юлией.
Юлии было интересно посмотреть мой проект как с наружи, так и изнутри. Кроме того,
ей было интересно подробнее узнать о различных реализациях и решениях, которые
я имплементировал.
Я рассказал о том, какой смысл несёт в себе repository pattern, как у меня настроен
router и все маршруты включая guard. Я показал и рассказал ей о том, как у меня работает
Firebase и Firestore.
По итогам нашего звонка я получил несколько хороших советов по-дальнейшему workflow
(что нужно делать вот сейчас, а что отложить).

## Что решили

- В первую очередь нужно заняться Jamendo API, так как там всё непросто
- Для поиска в Jamendo подумать над фильтром неликвидных названий
- Продолжать работу в том же темпе

## Поручения и зоны ответственности

- Jamendo API : Имплементировать модели и сервисы
- UI: Продолжать разработку UI компонентов

## Запланировано

- Создать всю инфраструктуру Jamendo API
- Создание сервисов для всех необходимых эндпоинтов
- По возможности добавлять изменения в UI
